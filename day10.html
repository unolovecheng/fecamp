이벤트 관련 함수

jQuery에서 이벤트를 추가하거나 제거할 때는 .on() 또는 .off() 함수를 사용한다. 임의로 마치 사용자가 이벤트를 발생시킨 것처럼 이벤트를 실행할 때는 .trigger() 함수를 사용한다. 앞서 배운 내용은 9일차 강의를 참고하세요.

애니메이션 관련 함수

  • .show()/.hide()/.toggle(): 대각선으로 펼쳐지며 나타나거나 접히며 사라진다. 시간을 뜻하는 인수가 없으면 display 속성을 none 또는 그 밖의 값으로 설정한다.
  • .fadeIn()/.fadeOut()/.fadeToggle(): 페이드 인/아웃 효과
  • .slideDown()/.fadeUp()/.fadeToggle(): 아래로 펼쳐지며 나타났다가 위로 닫히는 효과
  • .animate(): 사용자 정의 애니메이션을 작성한다.
  • .stop(): 진행 중인 애니메이션을 현재 상태 그대로 중지한다.
  • .finish(): 진행 중인 애니메이션을 즉시 완료한다.
  • .delay(): 애니메이션을 전달한 시간만큼 지연시킨다.
  • 애니메이션에 사용되는 시간 문자열 'fast'는 200ms를 뜻하고 'slow'는 600ms를 의미한다.

CSS 관련 함수

  • .css: CSS 속성의 값을 가져오거나 설정한다. 설정한 CSS는 인라인 스타일로 추가된다.
  • .addClass()/.removeClass()/.toggleClass(): 클래스를 추가하거나 제거한다.
  • .hasClass(): 인수로 전달한 클래스가 설정되어 있는지 확인한다.
  • .width()/.height(): 엘리먼트의 너비와 높이를 반환하거나 설정한다.
  • .innerWidth()/.innerHeight(): 엘리먼트의 너비와 높이를 반환하는데, 패딩은 포함하지만 마진은 포함하지 않는다. window와 document 객체에는 사용할 수 없다.
  • .outerWidth()/.outerHeight(): 엘리먼트의 너비와 높이를 반환하는데, 패딩과 마진을 다 포함한다.
  • .offset(): 엘리먼트의 위치를 가진 object를 반환한다. 이 객체에는 left, top 프로퍼티가 있는데 각각 전체 페이지의 왼쪽 끝과 위쪽 끝에서 멀어진 픽셀 거리를 나타낸다.
  • .position(): 엘리먼트의 위치를 가진 object를 반환한다. 이 객체에는 left, top 프로퍼티가 있는데 각각 부모 오프셋 객체의 왼쪽 끝과 위쪽 끝에서 멀어진 픽셀 거리를 나타낸다.
  • .scrollLeft()/.scrollTop(): 엘리먼트 내부의 스크롤 위치를 설정하거나 반환한다.
